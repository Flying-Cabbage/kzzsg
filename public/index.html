GitHub Actions 版说明

这个版本没有后端 API。GitHub Actions 会按工作流里的 cron 定时执行 scripts/generate_data.py，生成 public/data/bonds_latest.json，然后部署到 GitHub Pages。

页面上的“重新读取页面数据”只会重新拉取已部署的 JSON；真正刷新源数据，需要等定时任务，或在 GitHub 仓库 Actions 页面手动运行 workflow。

计算口径

1 手可转债按 10 张、1000 元面值估算；买入股数按 A 股 100 股整数倍向上取整。安全垫 = 预计转债收益 ÷ 正股买入金额，可理解为正股最多可承受的理论下跌幅。

“预估上市价”默认来自简化模型，真实操作建议结合市场情绪、正股波动、评级、规模和公告手动调整。页面只做测算，不构成投资建议。
